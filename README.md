# product-shots

> One product photo in. Your full e-commerce visual stack out.

**Open-source Claude Code skills that turn one product photo into a full set of e-commerce visuals** — main images, detail pages, multi-angle shoots, social posts, and ad creatives.

Built for cross-border e-commerce sellers who ship listings to Amazon, Shopify, TikTok Shop, and beyond. Drop these skills into Claude Code and run your entire product visual workflow with one command.

## What's coming

A focused, opinionated kit — not 30 generic design skills, but the 5 that actually carry a cross-border seller's daily visual workload:

| Skill | What it does |
|---|---|
| `main-image` | Compliant marketplace hero shots (white background, ratio, badges) |
| `detail-page` | A+ detail-page module set with cross-image consistency |
| `multi-angle` | One reference → 9-angle product shoot, identity-locked |
| `ad-creative` | Platform-native ad variants across 8 platforms × 21 formats |
| `social-post` | Feed / Story / Reel / Carousel with industry-aware styling |
| `render` | Unified image-gen engine: GPT image 2, Gemini, Flux — one API |
| `hub` | Intent-routing entry: ask once, dispatched to the right specialist |

All skills speak Claude Code's native skill protocol — no MCP server, no custom runtime.

## Why

Cross-border sellers run a brutal visual pipeline: one SKU needs 7 Amazon images + an A+ page + 9 multi-angle shots + 4 ad creative variants + 6 social posts, in the right sizes for 5 platforms. Lovart did this in a closed SaaS. We're shipping it as **open-source Claude Code skills** so it runs inside your existing terminal, composes with your other skills, and never holds your brand assets hostage.
